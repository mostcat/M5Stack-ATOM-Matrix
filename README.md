# BT Gravity KEYBORAD

通过重力方向操作可发送4种控制指令

显示屏有重力感应提示,点离中心越远代表发速命令越快

方向倾斜大小控制指令的发送频率,倾斜越大发送密度越高

按键切换是否敏捷模式

  敏捷模式:分3档,水平-无动作,倾斜一点,发送1次控制指令,倾斜很多,发送2次倾斜指令
  
  非敏捷模式:分2档,水平-无动作,倾斜很多,发送1次倾斜指令
